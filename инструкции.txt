Для запуска нужно минимум указать входящий файл и вторым параметром исходящий.
Пример:
java Main in.txt out.txt

Исходящий файл должен всегда идти последним аргументом.
Стандартные установки: строки как String, сортировка - в прямой последовательности
Ключи:
	-a ---- прямая последовательность сортировки
	-d ---- обратная последовательность сортировки
	-i ---- строки рассматриваются как целые числа
	-s ---- строки рассматриваются как String
